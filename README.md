# GigShield AI 🚀

## 📌 Problem

Delivery partners lose income due to external events like rain, heat, or curfews. They have no protection system.

## 💡 Solution

GigShield AI provides parametric insurance:

* Weekly subscription model
* Automatic claim triggering
* Instant payout simulation

## ⚙️ Features

* User Registration
* Dynamic Premium Calculation
* Weekly Policy Purchase
* Event Simulation (Rain)
* Automatic Claim Approval

## ▶️ Run Project

1. Clone repo
2. Open frontend/index.html in browser
